`cfpkit` builds lightweight, developer-focused, self-hosted applications for tech meetups and local community organizers. 

Our flagship project is [**cfpkit/web-app**](https://github.com/cfpkit/web-app) — a self-hosted community CFP (Call for Papers) platform that simplifies the entire talk submission, blind-review, and selection lifecycle.

### Key Features of cfpkit
*   **Customisable Forms:** Drag-and-drop form builder with conditional logic.
*   **Public CFP Pages:** Sessionize-style landing pages with countdown timers.
*   **Blind Review:** Server-enforced anonymised grading to prevent bias.
*   **Email Lifecycle Automation:** Custom transactional emails via Resend.
*   **Zero-Maintenance Deploy:** Instant spin-up via `docker compose up -d` with automated SSL via Caddy.

Learn more and deploy your own instance at the [**cfpkit/web-app**](https://github.com/cfpkit/web-app) repository.
